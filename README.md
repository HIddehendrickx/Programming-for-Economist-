# Programming-for-Economist-
Repository for our project on part-time work in the Netherland.
Our project looks at part-time work in the Netherlands: how it changed over
time, the difference between men and women, how the Netherlands compares to
other European countries, and what happened around 2021.

# Data
All data is from Eurostat (dataset lfsa_eppga). It is downloaded automatically
in the R code, so you don't have to download anything yourself.
